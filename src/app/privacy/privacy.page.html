Privacy & Security Data Security All inventory data is transmitted securely to the remote database using RESTful APIs over encrypted HTTPS connections. We do not store sensitive backend credentials locally on the device. User Privacy This application only collects and manages inventory data. It does not track user locations, access personal contacts, or collect unnecessary personal identifiable information (PII). Input Validation Strict form validation is implemented (e.g., number-only fields for quantity and price) to prevent injection attacks and ensure database integrity.
